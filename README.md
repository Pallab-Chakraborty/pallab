# 🖥️ Pallab Chakraborty — Personal Portfolio

> Live site: [pallab-chakraborty.github.io/pallab](https://pallab-chakraborty.github.io/pallab/)

A clean, responsive personal portfolio website built with pure HTML, CSS, and JavaScript — showcasing my skills, projects, and experience as a B.Tech CS Engineering student at Jawaharlal Nehru University, New Delhi.

---

## ✨ Features

- ⚡ Fully responsive — works on all screen sizes
- 🎨 Dark-themed, modern UI with smooth animations
- 📂 Sections: About · Tech Stack · Experience · Projects · Achievements · Contact
- 🖥️ Terminal-style hero section with JSON profile viewer
- 🔗 Direct links to GitHub, LinkedIn, and email

---

## 🛠️ Built With

| Technology | Usage |
|---|---|
| HTML5 | Structure & semantics |

## 🚀 Getting Started

Clone and open locally:

```bash
git clone https://github.com/Pallab-Chakraborty/pallab.git
cd pallab
# Open index.html in your browser
```

No build tools or dependencies required — it's pure HTML/CSS/JS.
