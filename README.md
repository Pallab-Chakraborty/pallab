# 🖥️ Pallab Chakraborty — Personal Portfolio

> **Live Site:** [pallab-chakraborty.github.io/pallab](https://pallab-chakraborty.github.io/pallab/)

A clean, responsive personal portfolio website built with pure **HTML5**, **CSS3**, and **JavaScript** — showcasing my skills, projects, and experience as a B.Tech CS Engineering student at Jawaharlal Nehru University, New Delhi.

---

## ✨ Features

- ⚡ **Fully Responsive** — Optimized for desktop, tablet, and mobile devices
- 🎨 **Modern Dark Theme** — Clean UI with smooth animations and transitions
- 📂 **Comprehensive Sections** — About · Tech Stack · Experience · Projects · Achievements · Contact
- 🖥️ **Terminal-Style Hero** — Interactive JSON profile viewer with typewriter effect
- 🔗 **Quick Links** — Direct connections to GitHub, LinkedIn, and email
- ⚙️ **Zero Dependencies** — Pure vanilla HTML, CSS, and JavaScript
- 🚀 **Fast Loading** — No build tools, frameworks, or external libraries
